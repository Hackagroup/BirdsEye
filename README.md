# Developers

<p>In order to run, use:</p>

```
# Clone Repository
git clone https://github.com/Hackagroup/Twittive
cd Twittive

# Run Setup
bash setup.sh
```

<b>IMPORTANT : React takes a while to start! Don't turn off in the middle of `npm start`.</b>
